open-falcon + centos7

发表于 2017-09-16

我是部署在一台centos7的测试机上，官方文档地址https://book.open-falcon.org/zh_0_2/index.html
我没有按照官方文档配置，按官网说的一台机器启动全部后端组件没有起完全，所以我采用的是分布式安装启动。

下载：

wget https://github.com/XiaoMi/open-falcon/releases/download/0.0.5/open-falcon-0.0.5.tar.gz -O open-falcon.tar.gz

解压

mkdir tmp tar -zxvf open-falcon.tar.gz -C ./tmp

####模板策略报警配置

#####cpu
cpu.idle:cpu处于空闲状态的比例
报警条件： all(#3)<5

cpu.user:cpu一共花了多少比例的时间运行在用户态空间或者用户进程
报警条件：all(#3)>50

#####df
df.bytes.free.percent/fstype=ext4,mount=/:硬盘空闲空间
报警条件：all(#3)<10
硬盘小于10%之后会报警

#####load
load.15min:15分钟内运行队列的平均线程数
报警条件：all(#3)>8

#####port
net.port.listen/port=9000:9000端口监视情况
报警条件：all(#3)==0

#####proc
proc.num/name=redis:redis进程监控
报警条件：all(#1)==0
