Hexo + GitHub Pages搭建博客

发表于 2017-07-13

准备环境：

  • Node.js
  • Git

ps:因为Hexo博客系统基于Node.js编写，所以环境需要安装Node.js

GitHub注册和配置

  1. GitHub注册： https://github.com/
  2. 创建仓库名，规则：yourname.github.io
    yourname 必须是你的用户名

安装Hexo

官方文档 https://hexo.io/zh-cn/

npm install hexo-cli -g

查看版本
hexo version

初始化
hexo init wliying.github.io cd wliying.github.io npm install
运行本地服务
hexo server or hexo s

本地服务访问地址http://localhost:4000

####Hexo 部署到 GitHub

项目的_config.yml配置文件中，修改Deployment
deploy: type: git repo: git@github.com:wliying/wliying.github.io.git branch: master
本地文件提交到GitHub Pages
`// 删除旧的 public 文件
hexo clean

// 生成新的 public 文件
hexo generate
或者
hexo g

// 开始部署
hexo deploye
或者
hexo d
`

我会使用hexo clean ； hexo d -g

最后访问 https://wliying.github.io
success！
