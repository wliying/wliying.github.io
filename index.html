open-falcon + centos7

发表于 2017-09-16

我是部署在一台centos7的测试机上，官方文档地址https://book.open-falcon.org/zh_0_2/index.html
我没有按照官方文档配置，按官网说的一台机器启动全部后端组件没有起完全，所以我采用的是分布式安装启动。

下载：

wget https://github.com/XiaoMi/open-falcon/releases/download/0.0.5/open-falcon-0.0.5.tar.gz -O open-falcon.tar.gz

解压

mkdir tmp tar -zxvf open-falcon.tar.gz -C ./tmp

####模板策略报警配置

#####cpu
cpu.idle:cpu处于空闲状态的比例
报警条件： all(#3)<5

cpu.user:cpu一共花了多少比例的时间运行在用户态空间或者用户进程
报警条件：all(#3)>50

#####df
df.bytes.free.percent/fstype=ext4,mount=/:硬盘空闲空间
报警条件：all(#3)<10
硬盘小于10%之后会报警

#####load
load.15min:15分钟内运行队列的平均线程数
报警条件：all(#3)>8

#####port
net.port.listen/port=9000:9000端口监视情况
报警条件：all(#3)==0

#####proc
proc.num/name=redis:redis进程监控
报警条件：all(#1)==0

Airdoc-Eye 适配Canon眼底照相机自动上传程序安装手册

发表于 2017-08-12

安装环境：

  • window7 32bit
  • .Net Framework4.5.2

安装步骤

  1. 点击setup.exe启动安装
  2. 依次点击下一步进行安装
  3. 安装路径下点击exe程序启动

程序说明

  1. 默认安装路径：C:\Program Files (x86)\AutoUpload\AutoUpload\
  2. 安装过程会写入注册表开机自启动，电脑中开着360安全卫士会提示风险，

  3. 安装完成后，打开安装路径，.config文件为程序配置文件
    配置文件中可更改的配置项在appSettings里的key&value,分别是有：

    • 数据库文件MDF的路径examDBPath, examexDBPath
    • API请求的URL地址API_DOMAIN
    • APP_ID机构账号
    • APP_KEY机构密码

    配置文件的connectionStrings标签是数据库的连接字符串，最重要的是MDF数据文件地址
